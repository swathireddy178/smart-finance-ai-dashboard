# 💳 Smart Finance AI Dashboard

AI-powered financial analytics dashboard built using Streamlit, Pandas, Plotly and LLM (Llama 3 via NVIDIA API).


## 🚀 Project Overview

Smart Finance AI Dashboard is a full-stack data science project that allows users to:

- Securely log in
- Analyze spending trends
- Visualize monthly & yearly expenses
- Identify highest spending categories
- Get AI-generated financial insights
- Interact with a smart AI financial assistant

This project demonstrates data analysis, visualization, authentication systems, and LLM integration.


## ✨ Features

🔐 User Authentication (SQLite)  
📊 Monthly & Yearly Spend Analysis  
🏆 Highest Spending Category Detection  
🔍 Category Deep Dive  
📈 Interactive Plotly Visualizations  
🤖 AI Financial Assistant (Llama 3 API)  
🎨 Premium Dark UI  


## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- SQLite
- Llama 3 (NVIDIA API)
- Scikit-learn

## ⚙️ How to Run Locally

1. Clone the repository:
   git clone https://github.com/yourusername/smart-finance-ai-dashboard.git

2. Navigate into project folder:
   cd smart-finance-ai-dashboard

3. Install dependencies:
   pip install -r requirements.txt

4. Create a `.env` file and add:
   NVIDIA_API_KEY=your_api_key_here

5. Run the application:
   streamlit run app.py
